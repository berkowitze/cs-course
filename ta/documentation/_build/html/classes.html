magic_update(func: Callable[[importlib._bootstrap.Rubric], None]) → None[source]¶

using a rubric updating function, changes a) the base rubric, and b) each extracted rubric. updated rubrics are checked for validity before file writing (as of after homework 8 2018 rip)

This is most easily done in asgnipython

Parameters:

func (Callable[[Rubric], None]) – a function that takes in a rubric, returns None, and mutates the rubric.
Raises:
  • TypeError – invalid input types
  • AssertionError – func improperly modifies rubrics

Example:

asgn = HTA_Assignment("Homework 6")
qn = asgn.questions[0]

def updater(rubric):
    items = rubric['rubric']['Functionality']['rubric_items']
    items[2]['options'][3]['point-val'] = 5

qn.magic_update(updater)
