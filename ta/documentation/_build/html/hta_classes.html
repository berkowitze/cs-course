  • handins (List[Optional[Handin]]) – List of Handins/None if no Handin for that question
  • login (str) – login of student generating report
  • soft (bool, optional) – whether or not to only return the report information, rather than writing it into files; defaults to True
  • overrides (bool, optional) – whether or not to generate override reports defaults to False
