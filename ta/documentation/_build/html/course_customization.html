course_customization.determine_grade(raw_grade: RawGrade, late: bool, asgn: Assignment) → Grade[source]¶

given a raw grade (i.e. category -> float dictionary), determine the student’s full grade (i.e. category -> string dictionary)

Parameters:
  • raw_grade (RawGrade) – a raw grade (defined in custom_types)
  • asgn (Assignment) – the Assignment for which to determine the grade
  • late (bool) – whether or not the student handed in the assignment late
Returns:

The grade to give the student (following spec from custom_types)
Return type:

Grade

Example:

>>> determine_grade({"Functionality": 20, "Design": 12}, "Homework1")
32
>>> determine_grade({"Functionality": 12, "Design": 4}, "Homework 6")
{"Functionality": "Check Plus", "Design": "Check Minus"}
