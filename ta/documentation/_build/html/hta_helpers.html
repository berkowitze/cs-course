hta_helpers.latest_submission_path(base: str, login: str, mini_name: str) → Optional[str][source]¶

Returns None if no submission for this student on this assignment, and the path of the student’s latest submission if they did submit

Parameters:
  • base (str) – handin base path (i.e. /course/cs0111/hta/handin/students)
  • login (str) – login of student to get submission path for
  • mini_name (str) – minified assignment name (i.e. homework3)
Returns:

None if login has no handin for mini_name, and the path of the latest submission for login if they do
Return type:

Optional[str]

example:

>>> latest_submission_path('/course/cs0111/hta/handin/students',
                           'eberkowi',
                           'homework4')
'/course/cs0111/hta/handin/students/eberkowi/homework4/5-submission'
>>> latest_submission_path('/course/cs0111/hta/handin/students',
                           'eberkowi',
                           'homework3')
None
