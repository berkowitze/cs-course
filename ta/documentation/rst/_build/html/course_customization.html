course_customization module¶

course_customization.full_asgn_name_to_dirname(asgn_name: str) → str[source]¶
Parameters:asgn_name (str) – full name of assignment
Returns:the string of the directory name to be used for that assignment. this directory name will be used for all grading app related filesystems
Return type:str

example:

>>> full_asgn_name_to_dirname("Homework 3")
    "homework3"
